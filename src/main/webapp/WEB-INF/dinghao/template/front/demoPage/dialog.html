dialog 配置参数演示

1. 调用方法

title：弹出提示的title参数
content：弹出提示的内容参数

        $.artDialog({
            title:'消息',
            content:'消息内容可以带html代码'
        })

2. 回调函数

cancel关闭
ok：确认
默认参数均为false

        $('#button2').artDialog({
            title:'消息',
            content:'消息内容可以带html代码',
            cancel:function(){
                alert(0);
                return false;
            },
            ok:function(){
                alert('成功回调函数');
                return false;
            }
        })
